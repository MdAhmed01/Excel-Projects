# Excel Data Analysis Projects

## Overview

This repository contains Excel-based data analysis and dashboard projects created to practice and demonstrate data analysis skills using real-world business datasets.

## Projects Included

### 1. Coffee Sales Analysis Dashboard

#### Objective

Analyze coffee sales data to identify customer purchasing patterns, product performance, and sales trends.

#### Skills Used

* Data Cleaning
* Pivot Tables
* Pivot Charts
* IF & IFS Functions
* XLOOKUP
* Index & Match cases
* Data Visualization
* Dashboard Design

#### Key Insights

* Identified top-performing coffee products
* Analyzed sales by roast type
* Evaluated customer purchasing behavior
* Created an interactive Excel dashboard

---

### 2. Bike Sales Dashboard

#### Objective

Analyze customer demographics and purchasing behavior to understand factors influencing bike purchases.

#### Skills Used

* Pivot Tables
* Pivot Charts
* Conditional Formatting
* Data Analysis
* Dashboard Development

#### Key Insights

* Analyzed bike purchases by age group
* Compared purchasing trends across income levels
* Studied customer demographics and buying patterns
* Developed an interactive dashboard for decision-making
